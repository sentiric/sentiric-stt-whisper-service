# 🎧 Sentiric Stt Whisper Service

Saf C++ tabanlı, GPU hızlandırmalı Konuşmadan Yazıya (STT) ve Duyuşsal Zeka (Affective) motoru.

## 🚀 Hızlı Başlangıç
```bash
docker compose up
```
